# 作业守护者 🎓

AI 驱动的儿童作业辅导系统 - iPad/手机 Web App

## 项目简介

作业守护者是一个运行在 iPad/手机浏览器中的智能辅导应用，通过 AI 技术帮助孩子：
- 📹 实时监控注意力状态，走神时语音提醒
- 🎤 语音提问，AI 智能解答
- 📸 拍照分析作业质量
- 📊 生成每日学习报告

## 功能特性

### 1. 守护模式
- ✅ 调用摄像头实时监控
- ✅ 使用 MediaPipe FaceMesh 检测人脸
- ✅ 分析头部姿态和眼部状态
- ✅ 走神超过 15 秒语音提醒
- ✅ 记录专注时长

### 2. 智能问答
- ✅ 按住说话进行语音提问
- ✅ 支持文字输入
- ✅ Claude AI 智能解答
- ✅ 语音播报答案
- ✅ 聊天记录保存

### 3. 作业分析
- ✅ 拍照识别作业内容
- ✅ AI 评价书写和正确性
- ✅ 给出改进建议
- ✅ 鼓励性评价

### 4. 学习报告
- ✅ 累计专注时长
- ✅ 走神次数统计
- ✅ 提问数量记录
- ✅ 作业质量评分
- ✅ 今日时间线

## 技术栈

- **前端**: 纯 HTML + CSS + JavaScript（无需构建工具）
- **AI**: Claude Sonnet 4 API
- **人脸检测**: MediaPipe FaceMesh（CDN）
- **语音**: Web Speech API（识别 + TTS）
- **PWA**: 支持添加到主屏幕

## 快速开始

### 1. 准备工作

确保你有：
- Mac 电脑（开发环境）
- iPad 或 iPhone（运行环境）
- Claude API Key（从 console.anthropic.com 获取）
- 两设备连接同一 WiFi 网络

### 2. 在 Mac 上启动

```bash
# 进入项目目录
cd homework-guardian

# 启动开发服务器
./start.sh
```

服务器启动后会显示访问地址，例如：
- 本机访问: http://localhost:8080
- iPad 访问: http://192.168.1.100:8080

### 3. 在 iPad 上访问

1. 打开 Safari 浏览器
2. 输入上方显示的 IP 地址
3. **首次使用**: 输入 Claude API Key
4. **允许权限**: 摄像头和麦克风（必须）
5. **添加到主屏幕**:
   - 点击分享按钮
   - 选择"添加到主屏幕"
   - 像原生 App 一样使用

### 4. 配置 API Key

**获取 Claude API Key:**

1. 访问 https://console.anthropic.com
2. 登录并创建 API Key
3. 在应用首次启动时输入

**API Key 保存位置:**
- localStorage（本地存储，不会上传）
- 可随时在浏览器控制台查看/修改

## 使用说明

### 守护模式

1. 点击底部"守护"标签
2. 允许摄像头权限（首次）
3. 摄像头自动启动，开始检测
4. 专注时显示绿色✅，走神时显示红色⚠️
5. 走神超过15秒触发语音提醒

### 智能问答

1. 点击底部"问问"标签
2. **语音提问**: 按住"按住说话"按钮，说出问题，松开发送
3. **文字提问**: 在输入框打字，点击"发送"
4. AI 回答会自动语音播报

### 作业分析

1. 在"守护"页面点击📸拍照按钮
2. 拍摄作业内容
3. AI 自动分析并给出评价
4. 结果在"问问"页面显示

### 学习报告

1. 点击底部"报告"标签
2. 查看今日学习数据
3. 浏览时间线事件
4. 需要时可点击"重置统计"

## 注意事项

### 关于 HTTPS

- **iOS 摄像头访问**: 必须使用 HTTPS 或 localhost
- **本地开发**: 使用 IP 地址访问时，iOS 可能阻止摄像头
- **解决方案**: 
  - 方案1: 使用 ngrok 等工具提供 HTTPS
  - 方案2: 在 Mac 上配置自签名证书
  - 方案3: 直接在 iPad 上使用 localhost（需安装 Shortcuts 等工具）

### 关于权限

首次使用时需要授权：
- ✅ 摄像头权限（必须）
- ✅ 麦克风权限（必须）
- ✅ 通知权限（可选，用于后台提醒）

### 关于字体

如果中文字体显示异常，可能是字体缺失。系统会自动回退到：
- macOS: PingFang SC
- iOS: PingFang SC
- Android: Noto Sans CJK

## 常见问题

**Q: 摄像头无法启动怎么办？**
A: 检查：1) Safari 权限设置 2) 使用 HTTPS 或 localhost 3) 设备摄像头未被其他应用占用

**Q: 语音识别不工作？**
A: 确保：1) 允许麦克风权限 2) 网络连接正常 3) 使用的是 Safari 浏览器

**Q: AI 回答失败？**
A: 检查：1) API Key 是否正确 2) 网络连接 3) API 配额是否用完

**Q: 为什么一直显示"未检测到"？**
A: 确保：1) 光线充足 2) 人脸在画面中 3) 摄像头未被遮挡

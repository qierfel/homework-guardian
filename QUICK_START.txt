============================================
  作业守护者 - 快速启动指南
============================================

📁 项目位置: /Users/lirui/homework-guardian

🚀 启动步骤:

1. 打开终端（Terminal）
2. 运行以下命令:

   cd /Users/lirui/homework-guardian
   ./start.sh

3. 服务器启动后会显示访问地址

4. 在 iPad Safari 中输入地址

5. 首次使用输入 Claude API Key
   获取地址: https://console.anthropic.com

============================================
  iPad 访问步骤
============================================

1. 确保 iPad 和 Mac 在同一 WiFi
2. 在 iPad Safari 打开服务器地址
3. 允许摄像头和麦克风权限
4. 分享按钮 → 添加到主屏幕
5. 开始使用
